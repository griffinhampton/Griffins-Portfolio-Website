# Griffin Hampton Portfolio Website

This repository contains the source code for my comprehensive personal portfolio website, showcasing both technical expertise and modern web development practices. The site features a complete desktop experience with advanced animations, a dedicated mobile version, and professional portfolio sections.

Built entirely with **HTML5, CSS3, and vanilla JavaScript**, this project demonstrates how modern web technologies can create an immersive, performant, and visually compelling user experience without heavy frameworks.

---

## 🌟 Features

### **Desktop Experience**
- **Advanced Canvas Animations**  
  - Multi-layered particle systems (dust, orbs, shooting stars, smokey glows)
  - Interactive 3D diamond clusters with mouse-responsive physics
  - Animated letter system with collision detection and respawn mechanics
  - Hardware-accelerated rendering using `requestAnimationFrame`

- **Modern UI/UX Design**  
  - Gradient backgrounds with dynamic color schemes
  - Custom hamburger navigation with smooth slide-in menu
  - Text effects including glowing animations and color gradients
  - Professional loading screens with animated elements

- **Interactive Elements**  
  - Mouse-responsive diamond physics with orbital mechanics
  - Hover effects and smooth transitions throughout
  - Dynamic background elements that respond to user interaction

### **Mobile-First Approach**
- **Dedicated Mobile Version** (`/mobile/index.html`)  
  - Touch-optimized interface with swipe gestures
  - Bootstrap-powered carousel for certifications
  - Streamlined navigation and content layout
  - Mobile-specific interactions and animations

### **Professional Portfolio Sections**
- **Certifications Carousel**  
  - Interactive Bootstrap carousel showcasing professional credentials
  - Microsoft and LinkedIn Learning certifications
  - Custom styling with company branding and hover effects
  - Touch-enabled swiping for mobile devices

- **Project Showcase**  
  - Detailed project pages for FNAD 2, FNAD World, and the website itself
  - Skills tagging with Wikipedia integration
  - GitHub repository links and live demos
  - Responsive image galleries

### **AI Integration**
- **Zapier Chatbot**  
  Embedded AI assistant for visitor engagement and inquiries

### **Enhanced Functionality**
- **Smart Device Detection**  
  Automatic redirection to mobile experience based on device capabilities
- **Form Handling**  
  Formspree integration for contact form submissions
- **SEO Optimization**  
  Proper meta tags, structured data, and accessibility features
